# Product Brainstorm Skill

![Иллюстрация скилла](assets/illustration.png)

Структурированный продуктовый брейншторм: SCAMPER, How Might We, отбор по Impact/Effort — ≥10 идей, Top-3 детально. Часть продуктового toolkit Personal Corp.

## Проблема

Брейнштормы дают либо мало безопасных идей, либо длинный неструктурированный список без ранжирования.

## Решение

Агент прогоняет фреймворки дивергенции, бросает вызов допущениям и выдаёт отфильтрованный список идей для PRD или приоритизации.

## Когда применять

- Застряли на retention или монетизации
- Исследуете новое направление продукта
- Перед квартальным планированием

## Установка

```bash
cp -r skills/pm-brainstorm ~/.claude/skills/
```

## Пример промпта

```text
Придумай идеи, как повысить retention на бесплатном тарифе.
```
