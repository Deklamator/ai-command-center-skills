# Product Brainstorm Skill

![Skill illustration](assets/illustration.png)

Structured product ideation with SCAMPER, How Might We, and Impact/Effort screening — 10+ ideas, Top-3 detailed. Part of the Personal Corp product management toolkit.

## Problem

Brainstorms produce either too few safe ideas or a long unstructured list nobody can rank.

## Solution

The agent runs divergence frameworks, challenges assumptions, borrows across domains, and outputs a screened idea list ready for PRD or prioritization.

## When to use

- Stuck on retention or monetization
- Exploring a new product direction
- Before quarterly planning

## Installation

```bash
cp -r skills/pm-brainstorm ~/.claude/skills/
```

## Example Prompt

```text
Brainstorm ways to increase retention for our free tier users.
```

Slash command: `/pm-brainstorm`
