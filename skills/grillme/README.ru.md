# grillme

GrillMe - скилл кристаллизации идей в методологии Персональной корпорации.

Он интервьюирует CEO, перед вопросами исследует локальный контекст проекта, автоматически ведёт локальное dossier, строит дерево решений, создаёт Mermaid-диаграммы и готовит preview GitHub Issue.

## Для чего нужен

Используй `grillme`, когда идея ещё не готова для PRD, маршрутизации, GitHub Issues или передачи исполнителю.

Скилл снижает "аишность": он заставляет неясные места превратить в конкретные решения CEO.

## Основное поведение

- Один вопрос за раз.
- Перед вопросом читать локальный контекст repo.
- Автоматически вести локальный markdown dossier.
- Строить дерево решений.
- Генерировать Mermaid-диаграммы.
- Готовить PRD handoff.
- Готовить GitHub Issue preview.
- Требовать approval перед любым GitHub write.

## Граница безопасности

Автоматически разрешено:

- только локальная markdown-запись

Автоматически запрещено:

- GitHub Issues
- GitHub Projects
- запуск executor
- создание PR
- merge
- deploy
