# PRD Skill

![Иллюстрация скилла](assets/illustration.png)

Генерирует структурированный PRD с шаблонами под тип продукта и 10-пунктовым чеклистом качества. Часть продуктового toolkit Personal Corp.

## Проблема

Идея фичи понятна в разговоре, но не готова для дизайна и разработки — нет целей, правил, acceptance criteria и аналитики.

## Решение

Агент превращает описание фичи в PRD для ревью: фон, цели, дизайн модулей, acceptance criteria в Given-When-Then, события аналитики и самопроверка.

## Когда применять

- Перед дизайном или планированием спринта
- После приоритизации — для top backlog
- Когда стейкхолдерам нужен общий документ требований

## Установка

```bash
cp -r skills/pm-prd ~/.claude/skills/
```

## Пример промпта

```text
Напиши PRD для магазина баллов лояльности. Целевые пользователи — участники мобильного приложения. Цель — повысить повторные покупки.
```
