# Competitive Analysis Skill

![Skill illustration](assets/illustration.png)

Multi-dimensional competitor study with feature matrix, SWOT, Porter's Five Forces, and differentiation strategy. Part of the Personal Corp product management toolkit.

## Problem

Competitive context is scattered across websites, demos, and opinions — hard to turn into product decisions.

## Solution

The agent produces a structured competitor comparison, SWOT, positioning map, and three-layer differentiation plan (catch up / differentiate / create new).

## When to use

- Before writing a PRD
- For fundraising or strategy decks
- When entering a crowded category

## Installation

```bash
cp -r skills/pm-competitive ~/.claude/skills/
```

## Example Prompt

```text
Analyze Slack, Teams, and Discord for our async team chat positioning.
```
