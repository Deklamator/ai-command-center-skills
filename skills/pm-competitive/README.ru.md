# Competitive Analysis Skill

![Иллюстрация скилла](assets/illustration.png)

Многомерный конкурентный анализ: feature-матрица, SWOT, 5 сил Портера и стратегия дифференциации. Часть продуктового toolkit Personal Corp.

## Проблема

Конкурентный контекст размазан по сайтам, демо и мнениям — сложно превратить в продуктовые решения.

## Решение

Агент выдаёт структурированное сравнение конкурентов, SWOT, карту позиционирования и трёхуровневый план дифференциации.

## Когда применять

- Перед написанием PRD
- Для fundraising или стратегических деков
- При входе в переполненную категорию

## Установка

```bash
cp -r skills/pm-competitive ~/.claude/skills/
```

## Пример промпта

```text
Сравни Slack, Teams и Discord для позиционирования нашего async team chat.
```

Slash-команда: `/pm-competitive`
