# User Feedback Analysis Skill

![Иллюстрация скилла](assets/illustration.png)

Классифицирует фидбек, кластеризует темы, анализирует sentiment и NPS, выдаёт Top-10 болей с рекомендациями. Часть продуктового toolkit Personal Corp.

## Проблема

Отзывы, тикеты и CSV-экспорты шумные — боли теряются в объёме.

## Решение

Агент классифицирует фидбек по 6 категориям, кластеризует темы, триангулирует источники и ранжирует боли.

## Когда применять

- После сбора VOC из отзывов или саппорта
- Перед приоритизацией бэклога
- Когда упал NPS и нужны темы

## Установка

```bash
cp -r skills/pm-feedback ~/.claude/skills/
```

Проанализируй эту таблицу отзывов и дай top pain points с рекомендациями.
